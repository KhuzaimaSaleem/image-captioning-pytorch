# 🖼️ Image Captioning AI

An end-to-end deep learning project that generates natural language 
descriptions for images using a CNN + Transformer architecture.

## 🧠 How it Works
- **CNN Encoder** — Pretrained ResNet50 extracts image features
- **Transformer Decoder** — Generates captions word by word using attention

## 📊 Results
| Epoch | Loss |
|-------|------|
| 1     | 4.32 |
| 10    | 2.88 |

## 🛠️ Tech Stack
- Python, PyTorch, Torchvision
- Gradio (Web App)
- Flickr8k Dataset (40,000 image-caption pairs)

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Enable T4 GPU runtime
3. Run all cells
4. Launch Gradio app and upload any image!
